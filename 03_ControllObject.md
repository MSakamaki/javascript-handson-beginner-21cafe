## 演算子

[MDNリファレンス＠演算子](https://developer.mozilla.org/ja/docs/Web/JavaScript/Guide/Expressions_and_Operators)

プログラムは、前の章で話した変数と、演算子を使い様々な結果を作り出します。

演算子には以下のようなものがあります。

 + 算術系
	+ ``+`` 加算を行います。
	+ ``-`` 減算を行います。
	+ ``*`` 乗算を行います。
	+ ``/`` 除算を行います。
	+ ``%`` 除算のあまりを算出します。

```javascript

var i = 10;
var j = 5;

console.log("加算", i + j );
console.log("減算", i - j );
console.log("乗算", i * j );
console.log("除算", i / j );
console.log("除算あまり", i % j );

```
